# Android Charity App - Helping Hand
An Android application developed during university studies where users can explore and learn about different charity organization and choose where they would like to donate.

## Technologies
- Java
- Android Studio
- InVision

## Development process
- Created user stories before development
- Designed a navigation flow diagram to map the application's structure
- Created wireframes in InVision before development
- Implemented the design in Android Studio

## Features
- Browse organizations
- Navigate through different screens
- User-friendly interface

## Purpose
University project created to practice Android Studio development and application design.
